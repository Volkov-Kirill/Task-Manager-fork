# Task Manager WPF

## 📌 Описание проекта

Приложение для управления задачами (Task Manager), разработанное на C# с использованием WPF.

## 🛠 Технологии

* C#
* WPF (MVVM)
* ADO.NET
* SQLite

## 📂 Структура проекта

* Models — модели данных
* Views — интерфейс (XAML)
* ViewModels — логика UI
* Services — бизнес-логика
* Data — работа с базой данных

## ⚙️ Функционал

* Создание задач
* Просмотр списка задач
* Редактирование задач
* Удаление задач
* Изменение статуса
* Категории задач
* Сортировка задач по категориям

### Система ролей

#### Руководитель
* Может создавать задачи
* Может редактировать задачи
* Может удалять задачи
* Может назначать задачи работникам
* Может создавать задачи для всех работников

#### Работник
* Может создавать свои задачи
* Может изменять и удалять свои задачи
* Может менять статус задач руководителя

### Аутентификация
* Вход в аккаунт
* Выход из аккаунта
* Разделение прав доступа по ролям

## 🗄 База данных

Используется SQLite и ADO.NET. Создание таблиц и работа с данными выполняются с помощью SQL-запросов.

Основные таблицы:
* Users
* TaskItems

## 🚀 Запуск проекта

1. Установите среду разработки Visual Studio (рекомендуется версия 2022 или новее) с поддержкой: .NET Desktop Development 
2. Клонируйте репозиторий или скачайте проект
3. Откройте решение: 
    Файл TaskManager.sln 
4. Запустите проект:
    Нажмите F5 или кнопку Start
